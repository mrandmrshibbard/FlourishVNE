## üé≠ Core Workflows

### Character Creation Made Easy
1. **Open Character Wizard** from any scene
2. **Choose character archetype** (hero, mentor, rival, etc.)
3. **Customize appearance** with layered sprite editor
4. **Define personality traits** and relationships
5. **Set initial stats and variables**

**Pro Tip**: Use the relationship mapper to visualize character connections!

### Building Interactive Choices
1. **Add choice command** to your scene
2. **Use Logic Builder** to create conditions:
   - Drag "If Variable" block
   - Connect to "Show Choice" block
   - Set variable name and value
   - Preview logic flow in real-time

3. **Test immediately** with Live Preview

### Managing Complex Variables
- **Relationship Panel**: See all variable connections
- **Usage Tracker**: Find where variables are used
- **Performance Monitor**: Keep stories running smoothly
- **Snapshot System**: Save variable states for testing
