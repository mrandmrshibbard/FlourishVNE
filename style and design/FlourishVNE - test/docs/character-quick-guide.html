üí° Pro Tips

  • Reuse sprites: If the body doesn't change, all expressions can use the same body sprite
  • Transparent PNGs: Always use PNG format with transparent backgrounds, not JPG
  • Same dimensions: All sprites for a character should be the same size (e.g., 1000x1500px)
  • Clear naming: Name sprites descriptively: "emma_body_casual.png", "emma_face_happy.png"
  • Test early: Create one expression fully, test it in a scene, then do the rest
  • Layer count: Simple characters need 1-2 layers, complex ones might use 4-5
üÜò Still Stuck?
If your character isn't showing up in scenes:
  • Check that sprites are assigned in Character Inspector
  • Make sure PNG files have transparent backgrounds
  • Verify z-index values are correct (lower = back, higher = front)
  • Try creating a simple test character with just 1 layer and 1 expression first
